为什么开始做个人网站？

昨天参加了阿里云 × Intel × SoloNest 联合举办的「一人公司 AI 实战工作坊」。到场大概 80 个人，开发者、设计师、产品经理、创业者都有。主办方的挑战很简单：用 AI，在 2 小时内做出你的个人网站。

实践流程

从 0 到上线，4 步搞定

Step 1
和 AI 一起梳理内容
先回答三个问题：你是谁、你在做什么、别人为什么要认识你。让 AI 帮你把脑子里的东西结构化——这才是最难的部分。
Step 2
让 AI 生成 HTML
把内容喂给 AI，生成单文件 HTML 页面。样式、排版、响应式全在一个文件里。你不需要懂一行代码。
Step 3
GitHub 托管代码
注册 GitHub 账号，创建仓库，把 HTML 推上去。免费的，也是你代码的家。
Step 4
Vercel 一键部署
Vercel 关联 GitHub 仓库，选目录，点 Deploy。30 秒后你就有了 https 在线地址。

我的 Prompt

怎么让 AI 帮我做网站

核心思路：先让 AI 理解你想表达什么，再让它生成页面。

Prompt 
请帮我创建一个个人网站，单文件 HTML，所有代码写在一个文件里。

关于我：
我叫 [你的名字]，做了 [X] 年 [职业/领域]，现在在 [你正在做的事]。
我相信 [一句你相信的话]。

页面结构：
1. 首屏：大字名字 + 一句话 tagline（中文斜杠风格，如：产品人 / AI 探索者 / 终身学习）
2. 关于我：3 个卡片，分别是「做过」「在做」「相信」
3. 我在做什么：2-3 个当前项目的卡片
4. 联系方式：邮箱 + 微信 + GitHub

设计风格：
- Apple / Linear 风格，浅色主题
- 大量留白，字体偏细，正文用灰色不是纯黑
- 卡片圆角，hover 时轻微上浮 + 阴影
- 移动端自适应

请直接输出完整的 HTML 文件，我可以保存后直接在浏览器打开。

以下是我的个人信息，请替换上面的占位符：
[在这里粘贴你的具体介绍，越详细越好]
工具越来越便宜。
内容越来越昂贵。
网站只是载体。
真正重要的是
你脑子里到底装着什么。
