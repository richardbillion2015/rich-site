实践流程

从 0 到上线，4 步搞定

Step 1
和 AI 一起梳理内容
先回答三个问题：你是谁、你在做什么、别人为什么要认识你。让 AI 帮你把脑子里的东西结构化——这才是最难的部分。
Step 2
让 AI 生成 HTML
把内容喂给 AI，生成单文件 HTML 页面。样式、排版、响应式全在一个文件里。你不需要懂一行代码。
Step 3
GitHub 托管代码
注册 GitHub 账号，创建仓库，把 HTML 推上去。免费的，也是你代码的家。
Step 4
Vercel 一键部署
Vercel 关联 GitHub 仓库，选目录，点 Deploy。30 秒后你就有了 https 在线地址。
我的 Prompt

怎么让 AI 帮我做网站

核心思路：先让 AI 理解你想表达什么，再让它生成页面。

Prompt 
// 占位：后续填入完整 Prompt

请帮我创建一个个人网站，要求如下：

角色：我是一个 [你的职业/身份]，目前在做 [你正在做的事]。

风格：
- Apple / Linear 风格，浅色主题
- 大量留白，克制、轻盈
- 移动端友好

页面结构：
1. 首屏：我的名字 + 一句话介绍
2. 关于我：背景和经历
3. 我在做什么：当前项目
4. 联系方式：邮箱和社交媒体

内容：
[粘贴你梳理好的个人内容]
工具越来越便宜。
内容越来越昂贵。
网站只是载体。
真正重要的是
你脑子里到底装着什么。
