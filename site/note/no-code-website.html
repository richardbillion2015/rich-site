缘起

为什么开始做个人网站？

昨天参加了阿里云 × Intel × SoloNest 联合举办的「一人公司 AI 实战工作坊」。到场大概 80 个人，开发者、设计师、产品经理、创业者都有。主办方的挑战很简单：用 AI，在 2 小时内做出你的个人网站。

实践流程

从 0 到上线，4 步搞定

Step 1
和 AI 一起梳理内容
先回答三个问题：你是谁、你在做什么、别人为什么要认识你。让 AI 帮你把脑子里的东西结构化地写出来——这才是最难的部分。
Step 2
让 AI 生成 HTML
把你整理好的内容喂给 AI，让它生成一个完整的单文件 HTML 页面。所有样式、排版、响应式适配都写在一个文件里。你不需要懂一行代码。
Step 3
GitHub 托管代码
注册一个 GitHub 账号，创建一个仓库，把你的 HTML 文件推上去。这是免费的，也是你代码的家。
Step 4
Vercel 一键部署
用 Vercel 关联你的 GitHub 仓库，选好目录，点一下 Deploy。30 秒后你就有了一个 https 的在线地址，可以发给任何人。
我的 Prompt

我是怎么让 AI 帮我做网站的

下面是我给 AI 的核心提示词。思路是先让 AI 理解我想表达什么，再让它帮我生成页面。你可以根据自己的内容替换其中的部分。

Prompt 
// 占位：后续填入完整 Prompt
// 你可以把自己的 Prompt 粘贴在这里

请帮我创建一个个人网站，要求如下：

角色：我是一个 [你的职业/身份]，目前在做 [你正在做的事]。

风格：
- Apple / Linear 风格，浅色主题
- 大量留白，克制、轻盈
- 移动端友好

页面结构：
1. 首屏：我的名字 + 一句话介绍
2. 关于我：我的背景和经历
3. 我在做什么：当前项目
4. 联系方式：邮箱和社交媒体

内容：
[粘贴你梳理好的个人内容]

完整的 Prompt 会在下一篇详细展开，包括如何迭代样式、处理响应式、添加动画效果等。
